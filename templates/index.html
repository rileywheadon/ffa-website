Flood Frequency Analysis Framework

This is a web interface for the flood frequency analysis (FFA) framework developed in Vidrio-Sahagun et al. 2024. The framework provides tools for both stationary and nonstationary FFA as well as exploratory data analysis. In addition to this website, the framework can also be accessed through a R package. For more information about the framework, see here.


To use this website, click on “Get Started” below. Then, you will be guided through the following five steps:

  • Dataset selection
  • Exploratory data analysis
  • Approach selection
  • Frequency analysis
  • Report generation
