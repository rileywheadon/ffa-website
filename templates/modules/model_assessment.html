Module Summary

This module computes multiple performance metrics and diagnostic indicators to assess the quality of a fitted flood frequency model. This includes accuracy (e.g., residual statistics), fitting efficiency (information criteria), and uncertainty (i.e., coverage-based metrics using confidence intervals).

Limited model assessment is available for subperiods that use a nonstationary approach, since plotting positions are not compatible with nonstationarity.
